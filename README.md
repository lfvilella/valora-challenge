# Valora Challenge
Valora's Company Selective process challenge.

Tech Stack:
- Python
- Django
- Django-REST
- Docker / docker-compose


# Running locally

    $ make build

    $ make createsuperuser

    $ make up

Optional:

    $ make test

Open admin: http://localhost:8000/admin
